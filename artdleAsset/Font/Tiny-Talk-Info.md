# Tiny Talk
Tiny Talk is a 4pt pixel art font designed to be as compact as possible while still being stylish and legible.

## Author
[VEXED](https://v3x3d.itch.io/)

## Basic Info
- Base Glyph Height: 4px
- Scales: 4, 8, 12, 16, etc
- Ascender Max: 3px
- Descender Max: 1px

## Kerning
None
