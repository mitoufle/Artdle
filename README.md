# Rising Mirmidon

Space opera tabletop RPG with DnD-inspired rules and a companion web app.

## Pillars
- Familiar DnD-style d20 core with futuristic gear and classes.
- Fast character creation with strong archetypes and customization.
- Encounter generator tuned for space opera threats and set pieces.

## Intended tooling
- Character builder
- Encounter generator
- VTT-friendly exports (tokens, stat blocks, JSON, printable sheets)

## Recommended stack
- TypeScript (web app)
- React + Vite (frontend)
- Node.js + TypeScript (API)
