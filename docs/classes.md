# Classes (Futuristic, DnD-Inspired)

Each class maps to familiar DnD roles but is distinct in theme and gear.

## Core classes
- Vanguard: front-line combatant, heavy armor, tactical maneuvers. (STR/CON)
- Star Knight: oath-bound guardian with energy blade and defensive auras. (CHA/STR)
- Psi Adept: full caster using psychic disciplines and battlefield control. (INT/WIS)
- Mechanist: tech-caster and builder with drones, gadgets, and upgrades. (INT)
- Hacker: infiltration, sabotage, precision strikes, cyber tricks. (DEX/INT)
- Envoy: social specialist, buffs, inspiration, diplomacy. (CHA)
- Marine: brutal striker, rage-like combat protocols. (STR/CON)
- Voidwalker: pact-like powers from unknown entities and void anomalies. (CHA)
- Pilot: scout and skirmisher, mobility, navigation, and vehicle mastery. (DEX/WIS)
- Medic: battlefield healer and support with biotech tools. (WIS/INT)

## Notes
- Subclasses will mirror DnD-style archetypes.
- Multiclassing can be allowed with standard DnD restrictions.
